# cookiecutter configuration

| Parameter               | Default                    | Description                                                                                                             |
|-------------------------|----------------------------|-------------------------------------------------------------------------------------------------------------------------|
| **project_name**        | ""                         | Name of your project e.g. `My New Project`, `Testing Stuff`, `my-project`                                               |
| **project_slug**        | `slug` from `project_name` | `URL`-friendly version of `project_name` used as the name for projects base directory and `library` name if applicable. |
| **license** | `MIT License`              | Open source license for the project visible in `LICENSE` file and source code comments.                                 |
| **license_owner**       | ""                         | Owner of the license (author/you?)                                                                                      |
| **cmake_min_version**   | `3.17`                     | Minimal `CMake` version required to run the project                                                                     |
| **cpp_standard**        | `C++17`                    | Version of `C++` standard                                                                                               |
| **clang_format_file**   | Nominal                    | Identifies the general zoningclassification of the sale.                                                                |
| **type**                | Nominal                    | Identifies the general zoningclassification of the sale.                                                                |
| **unit_testing**        | Nominal                    | Identifies the general zoningclassification of the sale.                                                                |
| **code_coverage**       | Nominal                    | Identifies the general zoningclassification of the sale.                                                                |
| **sanitize**       | Nominal                    | Identifies the general zoningclassification of the sale.                                                                |
| **include_what_you_use**           | Nominal                    | Identifies the general zoningclassification of the sale.                                                                |
| **with_ci**             | Nominal                    | Identifies the general zoningclassification of the sale.                                                                |
