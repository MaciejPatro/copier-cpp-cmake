# Getting Started

## What you'll need
#### Minimal requirements
- `Python` version `3.7` or newer.
- `cookiecutter` version `1.7.3` or newer.
- `cmake` with reasonably new version (`3.xx`)
```bash
python3 -m pip install cmake cookiecutter
```
#### Extensions
- `conan` for resolving `3rd-party library dependencies
```bash
python3 -m pip install conan
```

## Create a project from template:

To start project generation simply use command:
